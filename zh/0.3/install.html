安装¶

我们推荐的安装方法是使用 pip：

pip install releng-tool
 (or)
python -m pip install releng-tool

要验证是否已安装组件，可以使用以下命令：

releng-tool --version
 (or)
python -m releng --version

快速开始¶

以下我们提供了一系列步骤，以帮助你准备使用此组件的新环境。这个快速入门旨在让你使用最新版本的 Python。

linux¶

虽然 Python/pip 的使用在 Linux 发行版之间几乎是一致的，但以下是在 Linux 的特定发行版下安装此组件的一系列有用步骤。从终端开始，调用以下命令：

arch¶

$ sudo pacman -Sy
$ sudo pacman -S python-pip
$ sudo pip install releng-tool
$ releng-tool --version
releng-tool <version>

centos¶

$ sudo yum install epel-release
$ sudo yum install python-pip
$ sudo pip install releng-tool
$ releng-tool --version
releng-tool <version>

fedora¶

$ sudo dnf install python-pip
$ sudo pip install releng-tool
$ releng-tool --version
releng-tool <version>

opensuse¶

$ pip install releng-tool
$ releng-tool --version
releng-tool <version>

ubuntu¶

$ sudo apt-get update
$ sudo apt-get install python-pip
$ sudo pip install releng-tool
$ releng-tool --version
releng-tool <version>

os x¶

从终端开始，调用以下命令：

$ sudo easy_install pip
$ sudo pip install releng-tool
$ releng-tool --version
releng-tool <version>

windows¶

请先下载最新版本的 Python：

Python - 下载
https://www.python.org/downloads/

调用安装程序时，建议选择 ["Add Python to PATH"] 选项。但是，用户可以从绝对路径显式调用 Python （这些步骤的其余部分将假设 Python 在路径中可用）。

打开 Windows 命令提示符并调用以下内容：

> python -m pip install releng-tool
> python -m releng --version
releng-tool <version>

发展¶

要安装最新的开发源，可以使用以下 pip 命令：

pip install git+https://github.com/releng-tool/releng-tool.git
